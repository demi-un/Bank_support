бот: @UNIBANK_supportbot

### Описание
версия python: 3.11

##### Алгоритм работы:
- сообщение из бота находится в векторной базе 
данных chromadb при помощи семантического поиска
- в gigachat отправляется сообщение пользователя, 
контекст из базы данных и системное сообщение
- ответ гигачата отправляется пользователю
